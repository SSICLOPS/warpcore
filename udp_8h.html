void udp_rx ( struct warpcore *const  w,
struct netmap_ring *const  r 
)

Receive a UDP packet.

Validates the UDP checksum and appends the payload data to the corresponding w_sock. Also makes the receive timestamp and IPv4 flags available, via w_iov::ts and w_iov::flags, respectively.

The Ethernet frame to operate on is in the current netmap lot of the indicated RX ring.

Parameters
wWarpcore engine
rCurrently active netmap RX ring.

bool udp_tx ( const struct w_sock *const  s,
struct w_iov *const  v 
)

Sends a payload contained in a w_sock::ov via UDP.

For a connected w_sock, prepends the template header from w_sock::hdr, computes the UDP length and checksum, and hands the packet off to ip_tx(). For a disconnected w_sock, uses the destination IP and port information in the w_iov for TX.

Parameters
sThe w_sock to transmit over.
vThe w_iov to transmit.
Returns
True if the payloads was sent, false otherwise.
