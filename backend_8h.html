#define NUM_BUFS   900000

The number of extra buffers to allocate from netmap.

Extra buffers are buffers that are not used to support TX or RX rings, but instead are used for the warpcore w_sock::iv and w_sock::ov socket buffers, as well as for maintaining packetized data inside an application using warpcore.
